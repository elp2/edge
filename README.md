# Ed's Gameboy Emulator

## Instructions
* Install cmake and SDL-dev libraries
  * sudo apt-get install libsdl2-dev cmake
* mkdir build
* cd build
* cmake -DCMAKE_BUILD_TYPE=[Debug|Release] ..
